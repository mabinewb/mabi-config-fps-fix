마비노기 설정 파일 도구

Mabi FPS Fix Web

마비노기에서 내보낸 .muo 환경설정 파일의 DummyCharRenderModeFPS 값을 -1로 바꿔 드립니다. 수정된 파일은 환경설정 → 환경설정 관리 메뉴의 가져오기 버튼으로 적용할 수 있습니다.

간편 수정 브라우저 안에서 처리 가져오기용 MUO 다운로드

사용 방법

  1. 게임에서 환경설정 → 환경설정 관리 메뉴를 여세요.
  2. 내보내기 버튼을 누르고 환경설정만 체크하여 내보내세요.
  3. 내보낸 파일은 문서\마비노기\설정\목록 폴더에 저장됩니다.
  4. 이 페이지에서 내보낸 .muo 파일을 선택해 주세요.
  5. 수정 MUO 다운로드를 누르면 FPS 값이 -1로 바뀐 파일을 내려받습니다.
  6. 다운로드한 파일을 문서\마비노기\설정\목록 폴더로 이동하세요.
  7. 환경설정 → 환경설정 관리로 돌아가 가져오기 버튼을 누르고 수정된 파일을 선택하세요.
  8. 가져올 항목에서 환경설정만 체크한 뒤 확인을 누르세요.

파일 선택

내보내기·가져오기 파일 폴더
문서\마비노기\설정\목록
선택 파일 없음
현재 FPS 값 -

처리 상태

  • 대상 파일환경설정 내보내기 .muo
  • 변경 항목DummyCharRenderModeFPS
  • 적용 값-1
파일을 선택하시면 이곳에 처리 결과를 안내해 드립니다.

작업 로그

준비됨.

안내

  • 이 도구는 간소화 기능 시작 FPS 값을 -1로 바꾸는 용도입니다.
  • 파일이 이미 -1 상태이면 별도 다운로드 없이 안내해 드립니다.
  • 가져올 파일은 문서\마비노기\설정\목록 폴더에 있어야 합니다.
  • 내보내기와 가져오기에서는 환경설정 항목만 체크하는 것을 권장합니다.
  • 파일은 서버로 전송되지 않으며 브라우저 안에서만 처리됩니다.
