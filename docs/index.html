마비노기 설정 파일 도구

Mabi FPS Fix Web

마비노기 설정 XML 파일의 DummyCharRenderModeFPS 값을 -1로 바꿔 드립니다. 간소화 기능이 항상 작동하도록 적용할 때 사용할 수 있습니다.

간편 수정 같은 파일명 다운로드 설정 폴더에 덮어쓰기

사용 방법

  1. 먼저 게임 안에서, 바꿔도 괜찮은 아무 설정이나 한 번 변경한 뒤 저장해 주세요.
  2. 저장 후 %APPDATA%\Mabinogi\Setting 폴더로 이동해 주세요.
  3. 탐색기에서 수정된 시간 기준으로 정렬하시면 방금 저장된 XML 파일을 찾기 쉽습니다.
  4. 이 페이지에서 해당 XML 파일을 선택해 주세요.
  5. 수정 파일 다운로드를 누르시면 DummyCharRenderModeFPS가 -1로 바뀐 파일을 내려받을 수 있습니다.
  6. 다운로드한 파일을 같은 폴더에 다시 덮어써 주시면 됩니다.

파일 선택

기본 설정 파일 폴더
%APPDATA%\Mabinogi\Setting
선택 파일 없음
현재 FPS 값 -

처리 상태

  • 대상 파일마비노기 설정 XML
  • 변경 항목DummyCharRenderModeFPS
  • 적용 값-1
파일을 선택하시면 이곳에 처리 결과를 안내해 드립니다.

작업 로그

준비됨.

안내

  • 이 도구는 간소화 기능 시작 FPS 값을 -1로 바꾸는 용도입니다.
  • 파일이 이미 -1 상태이면 별도 다운로드 없이 안내해 드립니다.
  • 설정 파일은 보통 %APPDATA%\Mabinogi\Setting 폴더에 있습니다.
